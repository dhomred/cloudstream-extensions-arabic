# 🇸🇦 ملحقات CloudStream العربية

مستودع متكامل يحتوي على 48 مستخرج متقدم لمواقع عربية وعالمية، مع دعم كامل للمحتوى العربي والدولي.

## 📊 الإحصائيات
- **عدد المستخرجات**: 48 مستخرج
- **عدد المواقع المدعومة**: أكثر من 27 موقع
- **اللغات**: العربية والإنجليزية
- **أنواع المحتوى**: أفلام، مسلسلات، أنمي، برامج تلفزيونية

## 🚀 طريقة الإضافة إلى CloudStream

### الطريقة الأساسية (الموصى بها)
1. افتح تطبيق CloudStream
2. انتقل إلى **الإعدادات** ⚙️
3. اختر **مستودعات الملحقات** 📦
4. انقر على **+** أو **إضافة مستودع**
5. أدخل هذا الرابط:
   ```
   https://raw.githubusercontent.com/dhomred/cloudstream-extensions-arabic/main/repo.json
   ```
6. انقر **إضافة** ✅

### الطريقة البديلة (للتثبيت اليدوي)
1. قم بتنزيل ملفات `.cs3` من مجلد `builds/`
2. انسخ الملفات إلى مسار التطبيق:
   ```
   Android/data/com.lagradost.cloudstream3/files/Plugins/
   ```
3. أعد تشغيل التطبيق

## 🛠️ المميزات التقنية

### ✅ المميزات الأساسية
- **دعم متعدد الجودات**: 480p، 720p، 1080p، 4K
- **معالجة روابط مباشرة**: استخراج الروابط بسرعة وأمان
- **دعم الترجمات**: ترجمات عربية وإنجليزية
- **واجهة عربية**: دعم كامل للغة العربية
- **تحديثات تلقائية**: تحديثات دورية للملحقات

### 🔧 المميزات المتقدمة
- **معالجة روابط معقدة**: التعامل مع روابط مختصرة ومحمية
- **استخراج من JavaScript**: قراءة الروابط من أكواد البرمجة
- **التعامل مع الحماية**: تجاوز أنظمة الحماية البسيطة
- **بحث متقدم**: خوارزميات بحث محسنة
- **تخزين مؤقت**: تسريع التحميل عبر التخزين المؤقت
